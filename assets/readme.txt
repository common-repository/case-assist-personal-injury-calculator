== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/ca-personal-injury-calculator` directory, or install the plugin through the WordPress plugins screen.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Create a new or use an existing page / post. Then type the following shortcode in the provided text area of the page / post. Shortcode: [ca_personal_injury_calculator]
4. Save draft or publish the page / post.
5. Click preview or view the published page to see the plugin.

= Minimum Requirements =

* PHP 7.3 or higher. PHP 8.0 or higher is recommended.

= Manually =

== Frequently Asked Questions ==

= What Is a Personal Injury Calculator? =

A personal injury calculator. Is a tool that helps estimate potential compensation, for personal injury claims based on various factors, such as; medical expenses, property damage, and income loss.

= Can the Calculator Predict the Outcome of My Legal Case? =

No, the calculator provides an estimate and should not be considered a prediction of the actual legal outcome.

= Is My Information Secure When Using an Online Calculator? =

In the context of our "Case Assist Personal Injury Calculator" plugin. The information is handled on the client side and is not transmitted or stored on the server. Making it secure within the scope of the plugin.
